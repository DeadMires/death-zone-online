## AdMob note

AdMob usually checks `app-ads.txt` at the root of the developer website domain.

For `https://deadmires.github.io/death-zone-online/`, the domain root is:

`https://deadmires.github.io/app-ads.txt`

If AdMob does not verify the project URL, use a user GitHub Pages repository named `DeadMires.github.io` or connect a custom domain so `app-ads.txt` can be placed at the real root.

The language switcher is implemented in `assets/site.js`. Texts are grouped by language keys: `ru`, `en`, and `uk`.
